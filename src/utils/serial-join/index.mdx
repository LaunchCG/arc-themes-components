# Serial Join

The `serialJoin()` function takes in an array of values and returns it using the serial comma (Oxford comma) rules. It provides overrides for replacing the delimiter, conjunction, and spacer characters.

## Usage

```jsx
import serialJoin from "./serialJoin";

const formattedString = serialJoin(["one", "two", "three"]);
```

## Properties

| Name            | Description                                                                    | Default |
| --------------- | ------------------------------------------------------------------------------ | ------- |
| **list**        | An array of values that can be coerced to a string value or an array of nodes. | \[]     |
| **conjunction** | A set of characters that should be used for the "and".                         | "and"   |
| **delimiter**   | A set of characters that should be used for the "comma".                       | ","     |
| **spacer**      | A set of characters that should be used for the inserted spaces.               | " "     |
