# Serial Join

The `formatAuthors()` function takes in an array of author objects and returns an array of nodes of linked and unlinked authors (based on the data provided) using the serial comma (Oxford comma) rules.

## Usage

```jsx
import formatAuthors from "./formatAuthors";

const htmlString = formatAuthors(globalContent.credits.by);
```

## Properties

| Name        | Description                                                   | Default |
| ----------- | ------------------------------------------------------------- | ------- |
| **authors** | An array of `by-object`s from an `ans-item` `credits` object. | \[]     |
