# Attribution

The `Attribution` component is used to deonote a credit or an author of a piece of content. This accepts any content and/or links that you would like to represent the attribution.

## Usage

```jsx
import { Attribution } from "@wpmedia/arc-themes-components";

const Credit = () => <Attribution>By: Arthur Writer</Attribution>;
```
