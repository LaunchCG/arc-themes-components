# Separator

The `<Separator />` component is used to break up content. The `content` property can be set via `separatorString` prop.

The component is using a `<span>` that can be customized as a whole. The `content` property renders the `separatorString` prop, in this way. The default separator is set in Theme Settings.

## Usage

```jsx
import { Separator } from "@wpmedia/arc-themes-components";

const Feature = () => (
	<p>
		Everything needed: news <Separator /> Features <Separator separatorString="ðŸš¨" /> Breaking news.
	</p>
);
```
